Memory Allocation with Interference Graph:

A college project for my data structure class where I implemented register allocation by using a graph data structure. The algorithm checks if two data nodes exist at the same time by checking if they share an edge (interference). If they do, they will be assigned to different register, if they don't that means that the register is free for each of them and they can be assigned to the same register. Constraint Satisfaction Problem.
